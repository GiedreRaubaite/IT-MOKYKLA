Shopping Basket

Web application built with JavaScript and jQuery containing these features:

  • Two different view modes
  • Product list with image, name, description and available quantitys
  • Ability to add chosen items in the basket
  • Ability to put items out of the basket
  • Counting total amount and showing it in the bar menu
  • Showing confirmation modal when removing an item from shopping cart
  • Using localStorage

Try it out!

Lessons Learned

  • Local Storage property - setItem() and getItem() functions, serializing a JavaScript object into a JSON string
  • Higher-order functions - creating functions that operate on other functions, either by taking them as arguments or by returning them
  • Bootstrap 4 Modal Plugin and Tooltip - setting up and customizing
  • Clean code principles - clear descriptive naming, short functions, single-minded attitude

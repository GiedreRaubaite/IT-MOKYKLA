To Do App

Web application built with JavaScript containing these features:

  • Ability to add new Todo tasks
  • Ability to mark task as completed
  • Ability to clear all completed tasks
  • Ability to filter task by state: Done, Not done, All
  • Counter that shows how many tasks are remaining

Lessons Learned

  • Synchronous array methods - manipulating data using array element as a data storage
  • Higher-order functions - creating functions that operate on other functions, either by taking them as arguments or by returning them
  • Module Pattern in JavaScript - wrapping a set of variables and functions together in a single scope.
